추천 3신기

워모그의 갑옷(워모그) : 체력 600. 최대 체력 12% 획득. 벨트 + 벨트

구원 : 체력 150, 마나 15. 5초마다 주변 아군의 잃은체력의 15% 회복. 이후 5초동안 내구력 10% 획득. 벨트 + 눈물

가고일 돌갑옷(가고일) : 체력 100, 방마저 25. 상대하는 적 한명당 방마저 10 증가. 조끼 + 망토

용의 발톱(용발) : 마저 75. 최대체력 9%. 2초마다 최대체력의 2.5% 회복. 망토 + 망토

굳건한 심장(강심) : 체력 200, 방 20, 치명 20%. 내구력 8%, 체력이 50% 이상일 때 내구력 15%. 조끼 + 장갑

덤불 조끼(가갑) : 방 65, 최대 체력 5%. 평타피해 8% 감소. 기본공격으로 피해 받을 때 주변 적에게 마법 피해를 100 입힘. 쿨타임 2초. 조끼 + 조끼

수호자의 맹세(수맹) : 마나 30, 방 20. 최대 체력의 40%가 되었을 때 최대 체력의 25%에 해당하는 방어막과 방마저 20 5초간 획득. 조끼 + 눈물

적응형 투구(적투) : 마나 15, 주문력 10, 마저 20. 방마저 40, 기본공격 피격시 마나 1 회복. 망토 + 눈물

크라운 가드 : 체력 100, 주문력 20, 방어력 20. 전투 시작시 최대 체력의 25%에 해당하는 보호막 획득. 보호막 파괴시 주문력 25 획득. 조끼 + 지팡이

태양 불꽃 망토(태불망,썬파) : 체력 150, 방어력 20. 2초마다 2칸 이내의 적 하나에게 10초동안 1% 도트딜과 33%의 치감 부여. 조끼 + 벨트

저녁 갑주 : 체력 150, 마저 20. 전투 시작 후 10초간 방마저 25 증가. 2칸 내의 적 방깍 30%. 망토 + 벨트

이온 충격기(이온) : 체력 100, 주문력 15, 마저 25. 적이 스킬을 쓰면 사용 마나의 160%에 해당하는 마법 피해를 입음. 2칸 내의 적 마방깍 30%. 망토 + 지팡이
