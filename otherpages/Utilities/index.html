치유량 감소(치감)

태양불꽃 망토(썬파) : 방 20, 체력 150. 2초마다 2칸 내 적 한 명에게 10초동안 지속피해 1%, 치감 33%. 조끼 + 벨트

붉은 덩굴정령(레드) : 공속 35%, 피해량 증가 3%. 평타와 스킬이 5초동안 지속피해 1%, 치감 33%. 곡궁 + 곡궁

모렐로노미콘(모렐) : 체력 150, 주문력 25, 공속 10%. 평타와 스킬이 10초동안 지속피해 1%, 치감 33%. 벨트 + 지팡이

방어력 감소(방깎)

저녁갑주 : 체력 150, 마저 20. 2칸내의 적 방깍 30%. 전투 시작 후 10초동안 방마저 25 증가. 망토 + 벨트

최후의 속삭임(라위) : 공 15%, 공속 20%, 치명 20%. 물리 피해를 입힐 경우 3초동안 방깎 30%. 곡궁 + 장갑

마법 저항력 감소(마깎)

이온 충격기(이온) : 체력 100, 주문력 15, 마저 25. 2칸내의 모든 적 마깎 30%. 해당적이 스킬 사용시 마나 사용량의 160%에 해당하는 반사피해를 받음. 망토 + 지팡이

스태틱의 단검(스태틱) : 마나 15, 주문력 15, 공속 15%. 3번째 공격마다 적 4명에게 마법피해 35를 입히고 5초동안 마깎 30%. 곡궁 + 눈물
